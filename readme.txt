Simple share button without javascript. Easy to share and install.

== Description ==

FWD Social Share is a simple plugin that can be installed easily. The usefulness of this plugin is made
share button without using JavaScript, CSS and HTML only. The advantage is your website faster loading.

It also has a shortcode feature, so that you can put the share button anywhere in your article or content.

Currently, Social FWD Share supports 12 major social media.

== Installation ==

1. Upload `fwd-social-share` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `<?php fwd_social_share() ?>` in your templates
4. Place '[fwdss]' in your content (via WP editor)
